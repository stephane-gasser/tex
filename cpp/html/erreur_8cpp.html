◆ backerror()

void backerror ( const std::string &  msg,
const std::string &  hlp 
)

Back up one token and call error.

The back_error routine is used when we want to replace an offending token just before issuing an error message. This routine, like back_input, requires that cur_tok has been set. We disable interrupts during the call of back_input so that the help message won't be lost.

At certain times box 255 is supposed to be void (i.e., |null|), or an insertion box is supposed to be ready to accept a vertical list.

If not, an error message is printed, and the following subroutine flushes the unwanted contents, reporting them to the user.
