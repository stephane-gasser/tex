void backinput ( Token  t)

Undoes one token of input.

Sometimes TeX has read too far and wants to `‘unscan’' what it has seen. The back_input procedure takes care of this by putting the token just scanned back into the input stream, ready to be read again. This procedure can be used only if cur_tok represents the token to be replaced. Some applications of TeX use this procedure a lot, so it has been slightly optimized for speed.
